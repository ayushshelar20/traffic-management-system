About TrafficAI

A college mini project demonstrating how AI and real-time data can revolutionize urban traffic management.

📋 Project Description

TrafficAI is an AI-Based Smart Traffic Control System that combines real-world traffic data with visual simulation to demonstrate adaptive signal control. The system dynamically adjusts traffic signal timings based on real-time congestion levels and provides emergency vehicle priority override.

This project serves as a practical demonstration of how artificial intelligence can be applied to solve real-world urban traffic problems without requiring physical hardware or cameras.

🔀 Hybrid Approach

Real Data Component: The system is designed to fetch live traffic data from Google Maps Distance Matrix API, providing actual travel times and congestion information for real roads.

Simulation Component: A visual simulation layer renders an animated intersection with moving vehicles that respond to signal changes. When API data is unavailable, realistic random data is generated.

Why Hybrid? This approach ensures the demo works reliably for presentations while still demonstrating real-world API integration concepts.

🛠️ Technologies Used

HTML5 CSS3 JavaScript ES6+ Canvas API Chart.js Google Fonts CSS Animations LocalStorage API Intersection Observer

Architecture: Pure client-side application with no build tools required. Uses CDN-hosted libraries and ES6 features supported by all modern browsers.
